Tài Liệu API

  • v0Loader.show() Hiển thị loader mặc định
  • v0Loader.show("Văn bản") Hiển thị loader với văn bản tùy chỉnh
  • v0Loader.hide() Ẩn loader
  • v0Loader.setText("Văn bản mới") Cập nhật văn bản loader
  • v0Loader.setProgress(50) Đặt phần trăm tiến trình (0-50)
